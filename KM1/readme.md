Контрольное мероприятия заключается в решении задания по вскрытию шифртекста на базе шифра Вижинера  

Решение осуществляется с использованием приложенного кода python и теории, представленной на стр.45 книги "Криптоанализ классических шифров" (есть в репозитории).  

## Решение.
Предложенные для КМ тексты имеют 4х значный ключ, однако, по требованиям Евтеева, это необходимо доказать.  
Все тексты имеют разную длину, кратную 4.  
Необходимо представить текст в формате 4 столбцов, поочерёдно записывая по 4 символа в строку. Для этого можно воспользоваться [кодом div2columns](https://github.com/AronHopeless/MPEI_Crypto_Evteev/blob/main/KM1/div2columns.py)  
Скопируйте свой текст и поместите его в переменную `text`, затем просто запустите программу.

  <img src="https://github.com/AronHopeless/MPEI_Crypto_Evteev/blob/main/KM1/imgs/01.jpg" width=750>

Вывод программы разделён на 2 части.  
Первая часть - текст, представленный в формате столбцов.  
Следует перенести его в Excel файл ([образец Excel]() приведён в репозитории). Для этого необходимо:  
1. Выделить и скопировать вывод программы (только столбцы)
2. Перейти в Excel, выделить одну ячейку - Вставить
3. В верхней панели перейти на вкладку `Данные`, выбрать пункт `Текст по столбцам`
4. В открывшемся окне указать формат данных `с разделителями`, нажать Далее
5. Выбрать пробел в качестве символа-разделителя. Нажать Далее и Готово
6. Получен текст в 4 столбца

Вторая часть вывода программы - подсчёт количества повторений каждого символа алфавита в каждом из столбцов. Необходимо вручную перенести данные в соответствующие ячейки Excel таблицы. Блок таблицы называется "Частота появления букв в каждом столбце".

Для дальнейшего решения необходимо воспользоваться [кодом index](https://github.com/AronHopeless/MPEI_Crypto_Evteev/blob/main/KM1/index.py)  
Скопируйте свой текст и поместите его в переменную `ShifrText`, затем запустите программу.

<p align="left">
  <img src="https://github.com/AronHopeless/MPEI_Crypto_Evteev/blob/main/KM1/imgs/04.jpg" width=750>
</p>

Первым выводом программы являются рассчитанные для каждого столбца индекс совпадения. Если каждый из полученных индексов соответствует интервалу `0,053-0,07`, значит действительно существует чертырёхсимвольное кодовое слово.  

Далее осуществляется перебор взаимных индексов совпаднений для столбцов 1-2, 1-3 и 1-4. Если индекс взаимных совпадений попадает в интервал `0,053-0,07` - значит он искомый и его номер является необходимой величиной сдвига столбца 2, 3 или 4 относительно первого.  

Все основные значения рассчитаны, необходимо внести их в Excel-таблицу. Дальнейшие действия будут осуществляться в ней.  
Полученные сдвиги позволяют представить алфафит шифротекста так, чтобы выделить из него кодовое слово. Сначала записывается первый столбец алфавита, затем - со смещением столбцы. Смещение работает таким образом, что первый символ 2, 3 и 4 столбца записывается, начиная с вычисленного сдвига.

<p align="left">
  <img src="https://github.com/AronHopeless/MPEI_Crypto_Evteev/blob/main/KM1/imgs/05.jpg" width=500>
</p>

В представленном примере сдвиг 1-го столбца составляет 31, следовательно запись алфавита начинается с 31 клетки с буквы "А". Аналогично для столбцов 3 и 4.  
В полученной таблице необходимо найти строку, содержащуюю читаемое слово - оно является кодовым словом.  

Далее осуществляется расшифровка текста с помощью таблицы Вижинера. Вертикальный столбец таблицы соответствует кодовому слову, горизонтальный - исходному тексту. Символы в середине таблицы Вижинера являются символами шифртекста.

<p align="left">

</p>

Для расшифроки текста с использованием таблицы Вижинера необходимо начинать с символа кодового слова, в его строке отыскать символ шифртекста и взять первый символ столбца.  

<p align="left">

Повторяя данный алгоритм для каждого символа шифртекста, получим исходный текст.  
Также можно воспользоваться ресурсом [Vigenere Tool](https://www.boxentriq.com/code-breaking/vigenere-cipher) в режиме Custom.  
Введите шифртекст, установите кастомный алфавит: `АБВГДЕЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ_`, введите кодовое слово и нажмите `Decode`. Результат расшифровки можно скопировать для отчёта.
